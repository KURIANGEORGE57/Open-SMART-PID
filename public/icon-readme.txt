NOTE: For the Electron desktop app, you should place an icon.png file in this directory.
Recommended sizes:
- Windows: 256x256 or 512x512 PNG
- macOS: 512x512 or 1024x1024 PNG
- Linux: 512x512 PNG

For now, the app will work without a custom icon (it will use the default Electron icon).
You can use the favicon.svg as a starting point to create a PNG version.
